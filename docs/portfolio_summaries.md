## LinkedIn project summary

Built a portfolio-ready healthcare analytics platform called Smart Dataset Analyzer using Python, Streamlit, Pandas, Plotly, and scikit-learn. The app profiles arbitrary CSV/Excel datasets, activates only the analytics the data can support, discloses synthetic helper support transparently, and generates stakeholder-facing outputs across quality, readiness, healthcare intelligence, governance, and decision-support workflows.

## Interview-friendly verbal summary

I built Smart Dataset Analyzer as a product-style healthcare analytics platform rather than a single notebook or dashboard. It handles messy tabular data, scores what the dataset is actually ready for, uses transparent remediation or synthetic helper support when needed, and then layers in healthcare analytics, decision-support, governance, and export-ready stakeholder outputs. The goal was to demonstrate both analytics depth and product engineering discipline.
